특정 이미지를 가져와서 원하는 사이즈 만큼 캡쳐한 후 원하는 위치에 해당 이미지를 출력!!!

(이미지,캡쳐하고자 하는 X좌표, 캡쳐하고자 하는 Y좌표, 너비,높이,출력하고자 하는 X 좌표값, 출력하고자 하는 Y좌표값, 너비,높이)

1.export라는 명령어를 활용해서 js데이터를 출력하는 곳에서 2가지 선택지가 존재함!!!

-export "default" : 1개의 자바스크립트 안에서 1개의 단일 함수 혹은 변수 혹은 객체를 대표해서 출력하는 경우!!


-export : 1개의 자바스크립트 파일 안에 여러개 다수의 유틸리티용 함수 기능 탑재되어있는 경우!!! => JS => 복수 함수(*상황에 따라서 가져다가 사용)


- 2.import를 해야하는 스크립트 파일에서 2가지 방식으로 대응!!!
- 특정 변수 안에 고정된 객체의 값으로 가져오겠다는 의미!!!


-export: 해당 변수값으로 바로 찾아서 사용!!!
