****속성*****
- innerHeight  ; 실제 개발자가 작성하는 문서가 출력되어지는 높이 공간   _   순수 콘텐츠 공간. 상단 주소창, 툴바 제외
- innerWidth ; 실제 개발자가 작성하는 문서가 출력되어지는 너비 공간
-outerHeight ; 주소창+툴바의 공간까지 포함한 웹브라우저의 높이
-outerWidth ; 우측 스크롤바 까지 포함시킨 너비 의 공간
-pageXOffset ; 우리가 작성한 문서를 기준으로 얼만큼 X축으로부터 떨어졌는지에 대한 좌표값
-pageYOffset ; 우리가 작성한 문서를 기준으로 얼만큼 Y축으로부터 떨어졌는지에 대한 좌표값
-screenX ; 현재 사용자의 모니터를 기준으로 X 좌표
-screenY ; 현재 사용자의 모니터를 기준으로 Y좌표
-scrollX ; 현재 문서상에 X축으로부터 얼만큼 스크롤이 진행되었는지 체크할 수 있는 좌표값
-scrollY ; 현재 문서상에 Y축으로부터 얼만큼 스크롤이 진행되었는지 체크할 수 있는 좌표값
- localStorage ; 로컬 컴퓨터에 임시적으로 데이터를 보관시킬 수 있는 속성 (* 컴퓨터가 꺼지거나 웹브라우저가 새로고침되더라도 데이터를 보관) ex) 하드의 역할 
- sessionStorage ; 로컬 컴퓨터가 실행되고 있는 그 순간에만 일시적으로 데이터를 저장,보관 => 웹브라우저가 새로고침되거나 컴퓨터 종료 = > 삭제!! ex) 램의 역할
- location ; 특정 URL 페이지로 이동시키고자 할 때 사용하는 속성
- navigator ; 현재 접속한 로컬 컴퓨터의 기본 정보값을 확인하고자 할 때 사용하는 속성
