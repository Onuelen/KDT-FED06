HTML에 적용하는 3가지 방법

1) 인라인 스타일

2) 내부 스타일

3) 외부 스타일(*실무)


*CSS3 선택자 

1) 전체 선택자   =   "   *    " 
- wildcard 선택자
- reset css
-* == all


2) 태그 선택자
 - 특정한 태그를 선택해서 스타일 적용


3) 아이디 선택자
 - html 태그 중에서 id 속성값을 가지고 있는 태그들에게 스타일을 적용 - " # "
 - id 속성값은 반드시 html 에서 1번만 사용해야 한다.


4) 클래스 선택자
 - html 태그 중에서 class 속성값을 가지고 있는 태그 들에게 스타일을 적용
 - " . "
 - class 속성값은 html 에서 여러번 횟수제한 없이 사용가능!!
 - 반복적으로 공통된 스타일 적용


5) 후손 선택자
 - 부모 아래에 들어오는 모든 자식 요소들 
 - 자손 선택자도 모두 포함
 - " 띄어쓰기 " 


6) 자손 선택자
 - 부모의 직계 자식요소
 - " > "  사용 


7) 속성 선택자
  - 동일한 HTML 태그의 동일한 속성을 가지고 있는 경우, 안에 들어가있는 속성값에 따라서 스타일 상이하게 적용!
ex ) input [ type = " text " ] { }

8) 동위 선택자
 - 동 : 같을 "동"
 - 위 : "위" 치
 > 같은 위치에 속해있는 요소들 간 스타일을 적용 하고 싶을 때 => 반드시 형제 요소!!!
	" ~ "
> 특정 요소의  "바로 뒤에"   존재하는 형제 : " + "
> 특정 요소 뒤에 있는   "모든"   형제 : " ~ "


9) 구조 선택자_ 2가지

- 1) 일반 구조 선택자
 1개의 부모 요소 안에 있는 모든 형제들을 형태와 관계없이 입력된 순서대로 카운팅 !
nth-child(n)
홀수 or 짝수 번째 요소들만 찾아올 수도 있음
2n / 2n +1 


- 2) 형태 구조 선택자
 1개의 부모 요소안에 동일한 형태를 가지고 있는 형제들 가운데서 순서대로 번호를 카운팅 !
nth-of-type(n)
first-of-type
last-of-type


 * 문자 선택자 
 - 시작 문자 선택자
 > 특정 문서안에 시작하는 첫번째 단어 혹은 문장을 선택해서 스타일을 적용하고자 할때
::first-letter : 시작하는 첫 번째 문자열
::first-line : 시작하는 첫 번째 행

 - 전후 문자 선택자 (*중요)
> 문서 상에 특정 요소의 앞과 뒤에 어떤 스타일링을 적용하고자 할 때 사용할 수 있는 선택자
> 2차원적인 영역만 이야기 하는 것이 아님.
> 3차원적인 측면 (*이미지 혹은 아이템을 기준으로 z축으로 양수 / 음수 )
> 스타일을 적용할 때, 반드시 content ""

> position속석: 부모 relative / 자식 absolute
 부모요소의 전체 너비와 높이를 기준으로 자식요소의 위치를 결정
> 전,후 문자 선택자의 모체가 되어지는 요소는 부모요소
> 전 후 문자 선택자를 활용해서 탄생되는 요소는 자식요소 


*실무에서 전후문자 선택자를 프로페셔널하게 사용하는 방법!!
